Docker组件 (C/S架构)

  1. docker client ：Docker客户端
  2. docker server ：Docker daemon 主要组成部分 接收client 请求 并按照路由规则实现路由分发
  3. docker image : docker 镜像运行之后变成容器（docker run）
  4. docker registry: registry是docker镜像的中央存储仓库（pull /push）
